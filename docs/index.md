# Vindicta CLI

**Your command center for competitive Warhammer 40k.**

A powerful, Typer-based CLI for dice rolling, match tracking, and AI predictions — all from your terminal.

## Installation

```bash
uv pip install git+https://github.com/vindicta-platform/Vindicta-CLI.git
```

## Quick Examples

```bash
# Roll dice
vindicta dice roll 2d6

# Track a match
vindicta match new --player-list my.json

# Get predictions
vindicta oracle predict --my-list my.json
```

## Why CLI?

- **Speed** — No UI overhead
- **Scriptable** — Automate workflows
- **Portable** — Works anywhere Python runs
