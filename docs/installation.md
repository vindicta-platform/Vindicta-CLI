# Installation

## Requirements
- Python 3.10+
- uv (recommended)

## Install

```bash
uv pip install git+https://github.com/vindicta-platform/Vindicta-CLI.git
```

## Verify

```bash
vindicta --version
```

## Shell Completion

```bash
# Bash
vindicta --install-completion bash

# Zsh
vindicta --install-completion zsh

# PowerShell
vindicta --install-completion powershell
```
